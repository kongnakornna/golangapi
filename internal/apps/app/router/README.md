# ระบบเส้นทาง

```text
┌──────────────────────────────┐
│ มิดเดิลแวร์โกลบอล               │
│ RequestID/RealIP/Timeout/... │
└───────────────┬──────────────┘
                │
    ┌───────────┼────────────────┐
    │ Swagger   │ ตรวจสอบสุขภาพ  │
    │ /swagger  │ /health…       │
    └───────────┴────────────────┘
                │
             /api/v1
        ┌────────┴─────────┐
        │ เส้นทางสาธารณะ    │
        │ /auth/login      │
        │ /auth/refresh    │
        └────────┬─────────┘
                 │
           เส้นทางที่ได้รับการป้องกัน (JWT)
           /account/logout
           /users
```

## คำอธิบายเส้นทาง
- Swagger: `/swagger` และ `/swagger/*` ให้บริการอินเทอร์เฟซเอกสารและ JSON
- ตรวจสอบสุขภาพ: `/health`, `/health/detailed`, `/ready`, `/live`, `/version`, `/status`
- API v1: ภายใต้ `/api/v1` แบ่งออกเป็นเส้นทางสาธารณะและเส้นทางที่ได้รับการป้องกัน โดยเส้นทางที่ได้รับการป้องกันต้องใช้ JWT
