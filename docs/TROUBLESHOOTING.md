# การแก้ไขปัญหา
## การตั้งค่าไม่ได้ผล

- ตรวจสอบว่าไฟล์ `configs/config.yaml` มีอยู่หรือไม่
- ตรวจสอบว่า `CONFIG_PATH` ชี้ไปยังพาธที่ถูกต้องหรือไม่
- ตรวจสอบว่าคำนำหน้าตัวแปรสภาพแวดล้อมคือ ​​`APP_`

## การเชื่อมต่อฐานข้อมูลล้มเหลว

- ตรวจสอบการตั้งค่า `app.database`
- ตรวจสอบว่าสามารถเข้าถึงบริการฐานข้อมูลได้หรือไม่

## การเชื่อมต่อ Redis ล้มเหลว

- หากไม่จำเป็นต้องใช้ Redis ให้ตั้งค่า `APP_REDIS_ENABLED=false`
- เริ่ม Redis: `docker compose -f deploy/docker/docker-compose.yaml up -d`

## ความไม่สอดคล้องกันของ Swagger
- เรียกใช้ `./scripts/swagger.sh` เพื่อสร้างเอกสารใหม่

## พอร์ตซ้ำกัน
- แก้ไข `app.server.port` หรือปล่อยพอร์ตที่ใช้งานอยู่
